01. 平台基礎建設 (Infrastructure)

Frontend, Backend & Auth

前端應用層 (Frontend)

  • 框架: Next.js 提供 SSR/SSG，確保最高 SEO 權重。
  • 狀態管理: Zustand 或 Redux Toolkit。
  • UI: TailwindCSS 結合 Framer Motion 實作平滑動畫。

後端微服務 (Backend)

  • 核心 API: FastAPI (Python) – 極佳的非同步效能。
  • 任務佇列: Celery + RabbitMQ 處理耗時任務。
  • 資料庫: PostgreSQL (主資料) + Redis (快取)。

身份驗證 (Auth)

  • 機制: JWT 搭配 HttpOnly Cookies 防範 XSS 攻擊。
  • OAuth 整合: 支援 Google, GitHub 快速登入。
  • 權限控制: RBAC 分離學生、講師、管理員。

AI 自動化翻譯工作流

  • 原始內容撰寫以 繁體中文 為主。

03. 跨境與在地金流架構

Payment State Machine

針對東南亞市場，深度整合當地虛擬帳號與電子錢包。

支付網關動態路由

  • 台灣: 綠界 (ECPay) - 信用卡與超商。
  • 印尼/越南: Xendit - GoPay, MoMo 或虛擬帳號。
  • 國際: Stripe API，以 USD 計價。

交易狀態機 (Webhook)

  • PENDING: 等待非同步付款。
  • PAID: 收到 Gateway Webhook，解鎖課程。
  • FAILED: 超時未付款，釋放名額。

04. AI 助教與影音分發

RAG & Video CDN

HLS 加密串流架構

  • 儲存轉檔: AWS S3 搭配 MediaConvert 切片 HLS。
  • 內容保護: 結合 AES-128 加密，防堵直接下載。
  • 全球分發: 透過 CloudFront CDN 確保印尼/越南不卡頓。

RAG 課程助教

  • 資料向量化: 將影片逐字稿存入 Vector DB。
  • 精準問答: 自動帶入提問當下的時間段上下文。
  • 多語支援: 自動將外語提問對齊繁中知識庫並翻譯解答。
